Spreadsheet One Unit, Row Units and Column Units

In spreadsheets, the values in a row or a column often share the same unit, which we call one unit.
We say a spreadsheet has row units if the spreadsheet contains rows of one unit.
Similarly, we say a spreadsheet has column units if the spreadsheet contains columns of one unit.
